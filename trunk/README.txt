Ubicación de entregables
------------------------

El informe se encuentra en la carpeta Informe

La base de datos metadata se encuentra en src, archivo microstrategydata.mdb

El script de creación de tablas del sistema transaccional se encuentra en sql

Los archivos que contienen los datos de los sistemas transaccionales y
aquellos que generamos son src/db_original.rar y src/dates.csv

Los scripts de Pentaho de llenado de tablas se encuentran en src y tienen
extensión .ktr y .kjb

El modelo físico del Data Warehouse está en Informe/ModeloFisico.jpg
